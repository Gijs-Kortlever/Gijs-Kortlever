Project 1:  
Voor het allereerste project moesten wij individueel een lift etage maken, solderen, programmeren en de gegeven ontwerpen laser snijden.   
De lift werkte met een simpel nummer display voor de etage waarop de lift stond en 2 knoppen voor de richting, ook zaten er 2 led lampjes om aan tegeven of de lift was aangekomen.   De verschillende liftetages werkten op arduino uno's en communiceerden met een arduino mega via een I2C bus.

Project 2:  
Dit keer in duo moesten wij een rescue bot maken. Deze bot moest zelfstandig kunnen rijden aan de hand van zwarte lijnen en afgronden kunnen herkennen.  
Ook moest het bestuurbaar zijn via een app.   Bij dit project kregen we meer vrijheid en moesten we zelf de bot ontwerpen.   Ook begonnen we met het werken via scrum en 2 weken durende sprints.    Bij dit project heb ik de bot zelf ontworpen en de helft van de code geschreven.

Project 56:  
Eerste project met een opdrachtgever en een nuttig product.  
Bij dit project hebben ik en 3 andere projectgenoten een registratie systeem gemaakt voor de RDM Werkplaatsen op de RDM Campus van de Hogeschool Rotterdam.  
Het plan is dat dit door de Hogeschool zelf geÃ¯mplementeerd gaat worden.  
Wij hebben dan ook een prototype gemaakt.   
Dit registratie systeem bestaat uit een registratiepaal waar met de hogeschoolpas of de persoonlijke hogeschool QR-code ingecheckt kan worden, een database, API en PowerBI applicatie.  
Ik heb bij dit project gewerkt aan de database, API, server en natuurlijk de onderzoeken waarmee we ons uiteindelijke product hebben gevormd.   
De database is SQLite en de API heb ik gebouwd met Flask-Restful in python. Het draaide op een ubuntu server met een nginx proxyserver.
